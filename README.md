<h1>Pi-Ware</h1>
Pi-Ware is an app store inspired by Botspots <a href="https://github/com/Botspots/pi-apps">pi-apps</a> for the Raspberry Pi that allows you to install apps that you generally can't get from the official repos and/or without an extra amount of work.

# How to install and use Pi-Ware

## Install Pi-Ware
```sh
git clone https://github.com/piware14/pi-ware.git
cd pi-ware
bash install
```

## Update Pi-Ware
Pi-Ware will automatically fetch and install updates, and tell you to restart it to apply them.

If you widh to force update pi-ware, run
```sh
cd $HOME/pi-ware
bash updater
```

## Uninstall Pi-Ware
```sh
cd $HOME/pi-ware
bash uninstall
```
